## Description
This program creates a formatted application that is used to measure and store noise levels. The application features multiple user interfaces with interactive buttons that allow the user to input and store login, tenant, property manager, units, and reward information.

## Features 
- User authentication
- Interactive UIs
- Email Notification (Alert system)
- Graphs of noise levels
- Skillfully designed database

## Tech Stack
- React (Frontend)
- Node.js (Backend)
- PostgreSQL (Database)
- Bootstrap
- Docker Compose (Another way to control and run the frontend and backend server)

## Setup Instructions

### 1. [[Download Node.js | https://nodejs.org/en/download/current ]]

### 2. [[Clone the GitHub Repository | https://github.com/zboudreaux99/f25-Sapphire-5 ]]
git@github.com:zboudreaux99/f25-Sapphire-5.git

- Once the repository has been cloned use the command:
cd frontend

- then use the command:
npm run dev

to start the server for the website.

A local development URL should then be given through the terminal.
Example:  http://localhost:12345/

Copy this link and paste it in a browser to access the website. The page to choose wheather you are a tenant or landlord should be displayed.
